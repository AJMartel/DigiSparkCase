# Description:

  DigiSparkCase.scad
 
  Author: radiogeek381@gmail.com

  A box that can hold a single DigiStump DigiSpark arduino module.
  It is a clamshell box, with a slot in the bottom for in/out wires
  and a relief in the top for the LED.
